# TBR Games

Landing site for TBR Games — board games that help you pick what to read next.

Live at [tbr.games](https://tbr.games)

## Games

- **The TBR Dungeon** — explore a dungeon of books and build the ultimate reading list
- **The Dark TBR** — a darker challenge awaits those who dare to curate

## Local Development

```
bundle install
bundle exec jekyll serve
```

Site will be available at `http://localhost:4000`.
